# usdmxn-volatility-rf-4h

USDMXN volatility classifier for PyP Quant Mode.

This is an educational starter template for `USDMXN` on the `4h` timeframe. It implements the PyP Quant contract:

```python
train(data, config)
predict(model, market_data, config)
```

Use it as a baseline, then validate with PPE before any live deployment.

## Model

- Symbol: `USDMXN`
- Timeframe: `4h`
- Family: `sklearn RandomForest`
- Output: `UP`, `DOWN`, or `HOLD`
